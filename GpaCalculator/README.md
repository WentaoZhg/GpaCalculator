
============= Front-End ==================

<><> ----- Run ----- <><>
(1) Set port

.env
```
PORT=8081
```

(2) Project setup

In the project directory, you can run:

(3) npm install

or

$> yarn install


(3) Compiles and hot-reloads for development

$> npm start

or

$> yarn start


(4) Open http://localhost:8081 to view it in the browser.
